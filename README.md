## BitcoinDCA
% Author: Chris Evans

An open-source project to allow for non-custodial dollar-cost-averaging of Bitcoin using your Coinbase/Gemini Keys.

## Stack
 - Backend: Node-Express
 - Database: MySQL + Sequelize
 - Proxy-Server: NGINX
 - Frontend: React + Redux
 - Deployment: Docker
